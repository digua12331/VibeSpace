# 录演示视频 · HyperFrames

产品自闭环第 3 步：把刚实现的功能做成一段演示 MP4。数据契约见 `产品自闭环-总纲`。

## 前置依赖

- Node + npx
- FFmpeg（视频合成用）
- **首次运行 `npx hyperframes` 会下载几百 MB 的浏览器和依赖**，进度会看似卡住几分钟到十几分钟 —— 这是正常的，**不要误判成卡死去 kill 进程**，耐心等。

## 步骤

1. 写一段 HTML/CSS/GSAP 动画，演示本次为 issue `<N>` 新增/修复的功能（例如交互前后对比、一个新按钮的点击效果）。

2. 用 HyperFrames 渲染：

   ```
   npx hyperframes render
   ```

3. 把产出的视频放到约定路径（见总纲数据契约）：

   ```
   <项目根>/.aimon/artifacts/issue-<N>/demo.mp4
   ```

   如果 hyperframes 把视频输出在别处，把它移动/复制到这个路径。

## 失败处理

- FFmpeg 没装 → 告诉大哥需要先装 FFmpeg，不要绕开。
- 渲染报错 → 把 stderr 抛给大哥。
- 视频实在做不出来时不要卡死整条闭环：告诉大哥「视频步骤失败，原因 X」，并在第 4 步发 PR 时注明「演示视频缺失」，由大哥决定是否仍然发布。
