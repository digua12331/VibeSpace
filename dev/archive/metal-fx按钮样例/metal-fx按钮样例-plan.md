# metal-fx 按钮样例 · plan

## 大哥摘要
大哥说要用 `metal-fx` 这个包重构整个界面风格。我查清楚了:这个包**不是整套界面主题**,它只会给单个按钮加一圈"会动的金属光泽光环"(WebGL 渲染的发光金属边框特效),配不了整体配色和布局。所以**没法靠它一个包重构整个界面**。你选了"先看个效果再决定",所以这次我只做一件事:把左边项目栏底部那个「+ 新建项目」按钮接上这个金属特效,build 通过后你在浏览器里看实际长啥样,再决定要不要铺开或者换方向。改动很小、随时能撤回。

## 目标
- 在「+ 新建项目」按钮外面套一层 metal-fx 金属光环特效。
- 验收(浏览器可观察):打开 VibeSpace,左侧项目栏最底部的「+ 新建项目」按钮边缘出现一圈缓慢流动的金属反光/发光,其它按钮和界面风格不变。
- 工程验收:`pnpm -F @aimon/web build` 通过(含 TypeScript 类型检查)。

## 非目标
- 不改其它任何按钮、配色、布局、组件。
- 不做"整个界面换风格"(那是另一个大设计任务,不在本轮)。
- 不封装通用金属按钮组件、不做可配置开关(只用一次的抽象,不做)。

## 实施步骤
1. `pnpm -F @aimon/web add metal-fx`(已完成)。
2. 在 `ProjectsColumn.tsx` import `MetalFx`,把「+ 新建项目」按钮用 `<MetalFx>` 包起来,挑一个金属预设(silver/gold/chromatic 之一),pin `theme="dark"`(本应用是深色界面)。→ verify: build 通过。
3. build + 让大哥在浏览器里看效果。

## 边界情况
- 按钮原本有 `m-3` 外边距和虚线边框;metal-fx `normalizeHostStyles` 默认会抹掉子元素的 border/outline/box-shadow 换成金属环——这正是特效的预期表现。外边距移到 wrapper 上,保持原有间距。
- 特效用 WebGL,极老旧浏览器/禁用 WebGL 时可能不显示;本项目跑在本机现代浏览器,可接受。

## 风险与注意
- metal-fx 1.0.4 发布仅 3 周、单作者维护,属新包;本次只接一个按钮做评估,不深度依赖,风险可控。
- 若大哥看完不满意,删掉这层 wrapper + 卸载包即可完全回退。
